> 텍스트 스타일 관련 태그들을 알아봅시다. <b>태그는 문자를 진하게(bold) 변경합니다.

<strong>태그는 문자를 진하게(strong) 변경합니다.(권장)


<i>태그는 문자를 이탤릭체로 변경합니다.

<em>태그는 emphasis(강조)의 약어로 글자가 기울어져 나타납니다.(권장)

<cite>태그는 문자의 부분을 인용 할 때 사용합니다.(권장)

<u>태그는 글자에 밑줄선(underline)을넣을 때 사용합니다.


<s>태그는 글자에 취소선(strike though)을 그릴 때 사용합니다.

<strike>태그는 글자에 (strike though)을 그릴 때 사용합니다. (권장)

<sup>태그는 위첨자(superscript)를 표현합니다.

<sub>태그는 아래첨자(subscript)를 표현합니다.


<big>태그는 현재 기본 글자보다 좀 더 크게(+1)변경합니다.

<small>태그는 현재 기본 글자보다 좀 더 작게(-1)변경합니다.
